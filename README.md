# Café UNA — Frontend

## ¿Qué hace?

Es el sitio de CAFÉ UNA, donde los usuarios pueden conocer el café, ver productos, hacer pedidos, registrarse y ser voluntarios. También incluye un panel para quienes administran la página.

### Para usuarios

- **Inicio:** presentación del proyecto, fotos e información general.
- **Sobre nosotros:** historia y datos del equipo.
- **Productos:** catálogo con precios, fotos y opción de compra.
- **Carrito y checkout:** armar el pedido y completar la compra.
- **Voluntariado:** formulario para solicitar participar de voluntariado.
- **Perfil:** cada usuario puede ver y editar su información personal.
- **Inicio de sesión:** entrar con usuario y contraseña, recuperar contraseña si se olvida.

### Para administradores

Desde el panel de admin se puede gestionar casi todo el contenido del sitio:

- Textos e imágenes de la página principal.
- Información de “Sobre nosotros”.
- Inventario de productos (agregar, editar, quitar).
- Solicitudes de voluntariado (revisar, aprobar o rechazar).
- Usuarios registrados.
